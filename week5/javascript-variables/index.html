Variables

Vaiables are containers for storing data, javascript variables can be declared in 4 ways:

  1. Automatically
  2. Using the var keyword.
  3. Using the Let keyword.
  4. Using the Const keyword.

note: it is good practice to declare your variable before using them

Using var: var x=y; var x=5; note:strong> the var keyword was used in alll javascript code from 1995 till 2015, the let and const keyword where added to javascript in 2015. The var keyword should only be used for code written for older browsers.

Using let: let x=y; let x=5; only use let if you cant use const

Using const: const x=y; const x=5; use const if the value should not be changed, use it for an array or object

const price1 = 5; const price2 = 8; let total = price1 + price2;

where to use var, let or const. To declare a variable means to create the variable with the keyword

All javascript variables must be identified with unique names called identifiers.it can be short names like x and y or more descriptive names likes age, sum, total volume. Thee general rules for constructing variable names are:

  1. it can contain letters, digit, underscore and dollar sign
  2. it must begin with a letter.
  3. it can also begin with dollar sign and underscore but we will not use it in this tutorial.
  4. Names are case sensitive
  5. Reserved words like Javascript keywords cannot be used as a variable name.

The equal sign is an assignment operator, not an equal to, this is different from mathematics algebra. the equal to operator in mathematics is written as == in javscript.
